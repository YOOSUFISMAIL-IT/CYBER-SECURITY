This repository serves as a hands-on learning resource for understanding common attack vectors and implementing defensive mechanisms.

✨ Features
Password Attack Simulation (Attack.py) – Demonstrates brute-force attack concepts by testing a list of common passwords against a target credential

Log Analysis & Intrusion Detection (defense.py) – Monitors authentication logs to detect suspicious patterns and potential brute-force attempts

File Integrity Checking – Calculates SHA-256 hashes to verify log file integrity and detect tampering

Educational Examples – Simple implementations in C, C++, and Python for learning core programming and security concepts

🚀 Usage
Running the Attack Simulation
The Attack.py script simulates a dictionary attack against a hardcoded password:

bash

Copy

Download
python Attack.py
What it does:

Tests each password in a predefined list against the correct password ("python123")

Logs each attempt (success or failure) to access_log.tex

Generates a SHA-256 hash of the log file and saves it to log_hash.txt

Running the Defense Script
The defense.py script analyzes the logs generated by the attack simulation:

bash

Copy

Download
python defense.py
What it does:

Reads the previously generated log_hash.txt to verify file integrity

Scans access_log.tex for failed login attempts

Detects potential brute-force attacks (3 or more failed attempts)

Alerts if the log file has been modified
